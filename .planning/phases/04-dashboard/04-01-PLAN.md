**GET /data?window=30m** — last 30 minutes of checks + computed statuses per pair
```python
{
    "window": "30m",
    "checks": [{"node_ip": str, "target_ip": str, "ping_ok": bool, "ping_latency_ms": float|None,

    "statuses": [{"node_ip": str, "target_ip": str, "ping_status": "OK"|"Pending"|"NotAvailable",
                   "http_status": "OK"|"Pending"|"NotAvailable", "last_seen": float|None}],
    "timestamp": float
}
```

**GET /data?window=30d** — last 30 days of daily aggregated per-pair uptime
```python
{
    "window": "30d",
    "days": [{"date": str (YYYY-MM-DD), "connections": [
        {"node_ip": str, "target_ip": str, "total_checks": int,
         "ping_uptime_pct": float, "http_uptime_pct": float}
    ]}],
    "timestamp": float
}
```

**GET /node-list** — registered node IPs (matrix axes)
```python
{"nodes": [str, ...], "count": int}
```

**Status combination logic (from CONTEXT.md):** For the 30-minute view matrix, combine ping_status + http_status per pair:
- Both OK → OK (green #22c55e)
- Either NotAvailable → NotAvailable (amber #f59e0b)
- Both Pending (no data) → Pending (gray #9ca3af)
- Self-to-self diagonal → "—" (gray #d1d5db)

**30-Day View badge colors:**
- ≥ 99% uptime → #22c55e (green)
- ≥ 95% and < 99% → #f59e0b (amber)
- < 95% → #ef4444 (red)
- No data → #9ca3af (gray) with "—"

**3. Implement cached data fetching functions:**

```python
@st.cache_data(ttl=30)
def fetch_data_30m() -> dict | None:
    """Fetch 30-minute window data from leader. Returns None on failure."""
    try:
        resp = requests.get(f"{LEADER_URL}/data?window=30m", timeout=DATA_FETCH_TIMEOUT)
        resp.raise_for_status()
        return resp.json()
    except requests.RequestException:
        return None


@st.cache_data(ttl=30)
def fetch_data_30d() -> dict | None:
    """Fetch 30-day aggregated data from leader. Returns None on failure."""
    try:
        resp = requests.get(f"{LEADER_URL}/data?window=30d", timeout=DATA_FETCH_TIMEOUT)
        resp.raise_for_status()
        return resp.json()
    except requests.RequestException:
        return None


@st.cache_data(ttl=30)
def fetch_node_list() -> list[str]:
    """Fetch registered node list from leader. Returns empty list on failure."""
    try:
        resp = requests.get(f"{LEADER_URL}/node-list", timeout=DATA_FETCH_TIMEOUT)
        resp.raise_for_status()
        return resp.json().get("nodes", [])
    except requests.RequestException:
        return []
```

Each function uses `@st.cache_data(ttl=30)` per user decision D-05.
Error handling: catch `requests.RequestException` broadly, return None/empty-list for graceful degradation.

**2. Inside `render_tabs()` function:**

```python
@st.fragment
def render_tabs(data_30m, data_30d, nodes, leader_ok):
    tab1, tab2 = st.tabs(["30-Minute View", "30-Day View"])

    with tab1:
        _render_30m_view(data_30m, nodes)

    with tab2:
        _render_30d_view(data_30d, nodes)

    # Auto-refresh indicator
    _render_refresh_indicator(leader_ok)

    # Sleep 30s then fragment rerun
    time.sleep(30)
    st.rerun(scope="fragment")
```

**3. Implement `_render_30m_view(data_30m, nodes)`:**

- If `data_30m` is None or `statuses` list is empty:
  ```python
  st.markdown('<p style="color: #6b7280;">No data available for this time window</p>', unsafe_allow_html=True)
  ```

- Build a "combined status" lookup dict from `statuses`:
  ```python
  combined = {}
  for s in data_30m.get("statuses", []):
      src = s["node_ip"]
      tgt = s["target_ip"]
      ping = s["ping_status"]
      http = s["http_status"]
      # Combined: if both OK → OK; if either NotAvailable → NotAvailable; else Pending
      if ping == "OK" and http == "OK":
          combined[(src, tgt)] = "OK"
      elif ping == "NotAvailable" or http == "NotAvailable":
          combined[(src, tgt)] = "NotAvailable"
      else:
          combined[(src, tgt)] = "Pending"
  ```
- Also build a "latency" lookup from `checks` (latest check per pair):
  ```python
  latencies = {}
  for c in data_30m.get("checks", []):
      key = (c["node_ip"], c["target_ip"])
      # Keep the latest by timestamp
      if key not in latencies or c.get("timestamp", 0) > latencies[key].get("timestamp", 0):
          latencies[key] = c
  ```
- Sort `nodes` ascending for deterministic order.
- Render each source node as a `st.expander`:
  ```python
  for src_ip in sorted(nodes):
      # Build status summary for the expander header
      targets_ok = sum(1 for tgt in nodes if src_ip != tgt and combined.get((src_ip, tgt)) == "OK")
      targets_total = sum(1 for tgt in nodes if src_ip != tgt)
      if targets_ok == targets_total and targets_total > 0:
          summary = "All OK"
      elif targets_ok == 0:
          summary = "Pending"  # if no data for any target
      else:
          summary = f"{targets_total - targets_ok} of {targets_total} down"

      with st.expander(f"▶ {src_ip}  [{summary}]  —  {targets_total} targets"):
          # Sub-header row: matrix cells per target + detail columns
          cols = st.columns([2, 1, 1, 1, 1])
          cols[0].markdown("**Target**")
          cols[1].markdown("**Status**")
          cols[2].markdown("**Ping**")
          cols[3].markdown("**HTTP**")
          cols[4].markdown("**Last Check**")

          for tgt_ip in sorted(nodes):
              if src_ip == tgt_ip:
                  continue
              status = combined.get((src_ip, tgt_ip), "Pending")
              # Status badge color
              if status == "OK":
                  badge = f'<span style="background:#22c55e; color:white; padding:2px 8px; border-radius:4px; font-size:12px;">OK</span>'
              elif status == "NotAvailable":
                  badge = f'<span style="background:#f59e0b; color:white; padding:2px 8px; border-radius:4px; font-size:12px;">Not Available</span>'
              else:
                  badge = f'<span style="background:#9ca3af; color:white; padding:2px 8px; border-radius:4px; font-size:12px;">Pending</span>'

              check = latencies.get((src_ip, tgt_ip), {})
              ping_lat = f'{check.get("ping_latency_ms", "—"):.1f}ms' if check.get("ping_latency_ms") is not None else "—"
              http_lat = f'{check.get("http_latency_ms", "—"):.1f}ms' if check.get("http_latency_ms") is not None else "—"
              last_seen = datetime.fromtimestamp(check["timestamp"]).strftime("%H:%M:%S") if check.get("timestamp") else "—"

              cols = st.columns([2, 1, 1, 1, 1])
              cols[0].markdown(f'<span style="font-family:monospace; font-size:14px;">{tgt_ip}</span>', unsafe_allow_html=True)
              cols[1].markdown(badge, unsafe_allow_html=True)
              cols[2].markdown(f'<span style="color:#6b7280; font-size:14px;">{ping_lat}</span>', unsafe_allow_html=True)
              cols[3].markdown(f'<span style="color:#6b7280; font-size:14px;">{http_lat}</span>', unsafe_allow_html=True)
              cols[4].markdown(f'<span style="color:#6b7280; font-size:14px;">{last_seen}</span>', unsafe_allow_html=True)
  ```
