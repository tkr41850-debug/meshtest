<action>
Create `deploy/install.sh` with:

**Shebang:** `#!/bin/sh` (POSIX sh, no bashisms)

**Variables:**
```
MESH_STATUS_HOME="${MESH_STATUS_HOME:-$HOME/.local/meshtest}"
MESH_STATUS_VERSION="${MESH_STATUS_VERSION:-main}"
REPO_URL="https://github.com/opencode-ai/mesh-status.git"
```

**Prerequisite checks** (fail with actionable URLs before any FS ops):
- `command -v uv` → print "uv not found. Install: curl -LsSf https://astral.sh/uv/install.sh | sh"
- `command -v git` → print "git not found. Install: https://git-scm.com/downloads"
- `command -v curl` → print "curl not found. Install your distro's curl package"

**Install logic:**
- Parse flags: `-y`/`--yes` (skip prompts), `--help` (print usage)
- If `MESH_STATUS_HOME` doesn't exist: `git clone --depth 1 --branch "$MESH_STATUS_VERSION" "$REPO_URL" "$MESH_STATUS_HOME"`
- If it exists (reinstall): `cd "$MESH_STATUS_HOME" && git fetch --tags && git checkout "$MESH_STATUS_VERSION"`
- `cd "$MESH_STATUS_HOME" && uv sync`
- `cd "$MESH_STATUS_HOME"/frontend && npm ci && npm run build`
- Create sentinel file: `echo "$MESH_STATUS_VERSION" > "$MESH_STATUS_HOME/.mesh-status.install"`
- Generate `.env`:
```
# mesh-status configuration
# Generated by install.sh on $(date)
LEADER_URL=http://0.0.0.0:58080
LEADER_HOST=0.0.0.0
DATA_DIR=$MESH_STATUS_HOME/data
```
- Success banner:
```
────────────────────────────────────────────────
 mesh-status installed to $MESH_STATUS_HOME

 Start the leader:   $MESH_STATUS_HOME/start.sh --leader
 Start the node:     $MESH_STATUS_HOME/start.sh --node

 Dashboard:          http://localhost:58080

 To uninstall:       $MESH_STATUS_HOME/start.sh --uninstall
────────────────────────────────────────────────
```
</action>
