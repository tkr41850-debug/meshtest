    6. `test_run_check_cycle_semaphore_limits_concurrency(self)`:
       - Create `asyncio.Semaphore(2)` (low limit)
       - Patch `node.check_node` to introduce a small delay and track concurrency:
         ```python
         _concurrent = 0
         _max_concurrent = 0
         _lock = asyncio.Lock()
         
         async def mock_check(ip, timeout=5.0):
             nonlocal _concurrent, _max_concurrent
             async with _lock:
                 _concurrent += 1
                 _max_concurrent = max(_max_concurrent, _concurrent)
             await asyncio.sleep(0.05)
             async with _lock:
                 _concurrent -= 1
             return {"target_ip": ip, "ping_ok": True, "http_ok": True, "timestamp": time.time()}
         ```
       - Run with 6 peers
       - Assert `_max_concurrent <= 2` (limited by semaphore)
       - This test verifies the semaphore is actually restricting concurrency
