- CheckResult uses ping_ok/http_ok booleans (matching real API), not ping_status/http_status strings

- Two separate bar rows per pair (ICMP + HTTP) instead of single gradient bar
- 30-day view always expanded — flat layout with sticky source headers
- Gap between matrix and cards via Tailwind mb-6 class
- Backend infers node_ip by matching target_ip against _results keys
